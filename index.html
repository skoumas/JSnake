JSnake

Fully Customizable Snake Plugin for JQuery

A fully customizable JQuery plugin that implements a snake game. You can use it in loading screens, like a small nice widget for a website or develop your own snake game.

Express Install

To start implementing your JSnake object right away just add this JavaScript file into your main header. 
<!DOCTYPE html>
<html lang="en">
  
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>Your Title</title>
  </head>
  <body>
  
    <script src="http://code.jquery.com/jquery-1.11.2.min.js"></script>
    <script src="js/jsnake.js"></script>
    <div id="snake1"></div>

    <script>
        $("#snake").JSnake();
    </script>
  
  </body>
</html>

Setting up a new JSnake instance is super easy. Just add the JSnake command after the JQuery selector of the object or objects you want to create the new snake game.

The ideal tag to create a new JSnake instance is the div tag. The script will then create a canvas object inside your div

$("#screen").JSnake();

Customization

Setting Description Type Default
width The width of your JSnake instance in pixels Integer 450
height The height of your JSnake instance in pixels Integer 450
sound Enable this if you want sound in your snake Boolean True
glow Renders the screen with a glowing effect simulating an arcade screen. Integer 0
direction The direction which the snake moves. Controlled by the keyboard's arrow keys String ENum "right","left","up","down" Right
gameSpeed The speed of the game expressed in milliseconds. The main game loop runs with this value Integer Right
snakeLength The length that the snake has when starting the game. Integer Right
startingScore A starting point for the score Integer Right
resolution The resolution of the game expressed as Width/Resolution. The more the detail and space for the snake to move the higher the value. Integer 20
showScore Shows the current player's score if enabled Boolean True
walls Game over when the snake head meets the end of the screen Boolean True
eatSelf Game over when the snake head meets it's body Boolean True
showStart Shows up a starting screen at the beginning Boolean True
startPosition The starting position of the snake at the start of the game Array {x,y} {x:0,y:0}
Colors Customizing the game's colors. Array See Bellow

Custom Colours

Colour Name Description Default
back The colour of the screen #ffffff
snake The colour of the snake #000000
food The colour of the food #000000
text The colour of the screen #000000
wall The color that the borders will have #000000

Events

You can apply your own function when a specific event takes place in JSnake such as when the game starts or when the snake hits the wall

Event Description Returns
onStartScreen When the starting screen load.
onStart When the game starts.
onLoop Called at each game circle.
onPause When the gameplay pauses.
onEatSelf When the snake eat it self. (score,gameovers)
onHitWall When the snake hits the wall. (score,gameovers)
onGameOver When the game is over. You can use this function to combine the onEatSelf and onHitWall events. (score,gameovers)

Functions

You can control your JSnake instance by a set of external functions. The functions are listed below.

To use those function simply call the function name after the created JSnake instance as shown below

Function Description Arguments
setDirection Sets the direction of the snake Direction("right","left","up","down")
setScore Changes the current score Integer
beep Generates a beeping sound null
togglePause Pauses and un-pauses gameplay null 
var snake=$("#yourdiv").JSnake();

$("#beepButton").click(function(e){     
    snake.beep();
});

$("#scoreButton").click(function(e){    
    alert( "The score is : " + snake.getScore());
});

$("#togglePauseButton").click(function(e){
    snake.togglePause();
});
